Questions from the counter

I don't know what to cook tonight. Where do I start?

Type in whatever's actually in the fridge — half-used, odd-shaped, doesn't matter. FridgeAI matches it against real dishes instead of a recipe box you have to shop for first.

How does the matching actually work?

Your ingredient list goes to Gemini with instructions to weigh flavour pairing, method, and what's realistically missing, then it returns a match score per dish so you can see the gap before you cook.

Is the meal wheel a real 7-day plan or just decoration?

It's a real plan — each day is generated for your chosen diet and household size, you're just moving through it as a wheel instead of a spreadsheet grid.

Do I need my own API key?

Yes — this build calls your own Gemini key from a small server so the key never sits in the browser. Add it to .env and restart the server.
